## 📚 Claude Codeとは？

Claude Codeは、**ターミナルで動作するAIコーディングアシスタント**です。

### 特徴

- 🖥️ **ターミナルベース** - エディタに依存せず、既存のワークフローに統合
- 🤖 **実行能力** - ファイル編集、コマンド実行、Gitコミットまで自動化
- 🧠 **コンテキスト理解** - プロジェクト全体を把握して適切な提案
- 🔧 **カスタマイズ可能** - プロジェクトメモリで独自のルールを設定

### 他のAIツールとの違い

#### エディタベース vs ターミナルベース

```
GitHub Copilot  → エディタ内のコード補完
Cursor          → エディタ内の対話とコード編集
Claude Code     → ターミナルでファイル編集・コマンド実行・Git操作まで完結
```

#### CLI AI ツール詳細比較

| 機能 | GitHub Copilot CLI | Gemini CLI | Claude Code |
|------|-------------------|------------|-------------|
| **実行環境** | ターミナル | ターミナル | ターミナル |
| **ファイル編集** | ✗ | △（提案のみ） | ✓（直接編集） |
| **複数ファイル編集** | ✗ | ✗ | ✓ |
| **コマンド実行** | ✓（提案） | ✓（提案） | ✓（実行） |
| **Git操作** | ✗ | ✗ | ✓ |
| **プロジェクトメモリ** | ✗ | ✗ | ✓（.claude/CLAUDE.md） |
| **外部連携（MCP）** | ✗ | ✗ | ✓ |
| **コンテキスト理解** | 単発 | 単発 | プロジェクト全体 |
| **サブエージェント** | ✗ | ✗ | ✓ |

**Claude Codeの決定的な違い**:
1. **実行能力** - 提案だけでなく、実際にファイルを編集・コマンドを実行
2. **プロジェクトメモリ** - `.claude/CLAUDE.md`で独自ルールを永続化
3. **チーム共有** - プロジェクトメモリをGitで共有してワークフロー統一

## 🚀 インストール

### macOS / Linux

```bash
# ネイティブインストール（推奨）
curl -fsSL https://claude.ai/install.sh | bash
```

### Windows

```powershell
# PowerShell
irm https://claude.ai/install.ps1 | iex
```

### npm経由（どのOSでも）

```bash
npm install -g @anthropic-ai/claude-code
```

## 🎯 初めての3ステップ

### Step 1: Claude Codeを起動

```bash
# プロジェクトディレクトリに移動
cd your-project

# Claude Codeを起動
claude
```

初回起動時は認証が求められます。指示に従って認証してください。

### Step 2: 質問してみる

```
プロンプト: このプロジェクトは何をするものですか？
```

Claude Codeがプロジェクト全体を分析して説明してくれます。

### Step 3: 簡単な変更をしてみる

```
プロンプト: README.mdに「Hello, Claude Code!」という見出しを追加して
```

Claude CodeがREADME.mdを編集してくれます。

## 💡 よく使うコマンド例

### プロジェクトの理解

```bash
claude "このプロジェクトの構造を説明して"
claude "認証の仕組みを説明して"
```

### コード修正

```bash
claude "このエラーを修正して: [エラーメッセージ]"
claude "test_user.pyにテストを追加して"
```

### Git操作

```bash
claude "変更をコミットして"
claude "わかりやすいコミットメッセージで"
```

### ファイル参照

`@`記号で特定のファイルを参照できます。

```bash
claude "review @src/auth.js のセキュリティ"
```

## 🎨 便利な使い方

### 対話モード vs 単発実行

```bash
# 対話モード（継続的に質問）
claude

# 単発実行（1回だけ）
claude "タスクの内容"
```

### ヘルプとアップデート

```bash
# ヘルプを表示
claude --help
/help

# アップデート確認
claude update

# 環境診断
claude doctor
```

## ⚠️ 注意点

### セキュリティ

- **APIキーやパスワード**をコードに含めないように注意
- AIが生成したコードも**必ず人間がレビュー**
- 機密情報が含まれないか確認

### コスト

- Claude Consoleでの利用には課金が必要
- Claude Pro/Maxプラン、またはエンタープライズプラットフォーム（Bedrock/Vertex AI）

### 技術的制約

- **インターネット接続**必須
- 大規模なコードベースではコンテキストウィンドウに制限がある
